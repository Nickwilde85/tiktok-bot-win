# 🪟 Инструкция для Windows

## 🚀 Быстрая установка

### Вариант 1: Автоматическая установка (рекомендуется)
1. Запустите `install.bat` (двойной клик)
2. Дождитесь завершения установки
3. Если FFmpeg не установился автоматически, запустите `install_ffmpeg.bat`
4. В открывшемся блокноте добавьте ваш `BOT_TOKEN`
5. Запустите `start_bot.bat`

### Вариант 2: Ручная установка
```cmd
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
notepad .env
```

## ▶️ Запуск бота

### Простой запуск
Двойной клик на `start_bot.bat`

### Запуск из командной строки
```cmd
venv\Scripts\activate
python bot.py
```

## 🔄 Автозапуск при старте Windows

### Автоматическая настройка
Запустите `setup_autostart.bat` (может потребоваться запуск от имени администратора)

### Ручная настройка
1. Нажмите `Win + R`, введите `taskschd.msc`
2. Создать задачу → Общие:
   - Имя: `TikTokVideoBot`
   - Выполнять независимо от входа пользователя
3. Триггеры → Создать:
   - Начать задачу: При входе в систему
4. Действия → Создать:
   - Программа: `C:\путь\к\проекту\start_bot.bat`
   - Рабочая папка: `C:\путь\к\проекту`
5. Условия:
   - Снять галочку "Запускать только при питании от сети"
6. ОК → Сохранить

## 🛑 Остановка бота

### Простой способ
Запустите `stop_bot.bat`

### Через диспетчер задач
1. `Ctrl + Shift + Esc`
2. Найдите процесс `python.exe` с `bot.py`
3. Завершить задачу

## 📋 Управляющие файлы

| Файл | Описание |
|------|----------|
| `install.bat` | Автоматическая установка всех зависимостей |
| `install_ffmpeg.bat` | Установка FFmpeg (для YouTube) |
| `start_bot.bat` | Запуск бота |

## ⚙️ Требования

- Windows 10/11
- Python 3.9 или выше
- FFmpeg (для YouTube)
- Интернет-соединение

### Установка Python
1. Скачайте с [python.org](https://www.python.org/downloads/)
2. При установке отметьте "Add Python to PATH"
3. Перезагрузите компьютер

### Установка FFmpeg
FFmpeg необходим для скачивания видео с YouTube.

Запустите `install_ffmpeg.bat` и выберите способ установки:
1. Автоматическая через winget (Windows 11)
2. Автоматическая через Chocolatey
3. Ручная установка
4. Портативная версия в папку проекта

## 🔧 Решение проблем

### "Python не найден"
- Установите Python и отметьте "Add Python to PATH"
- Перезагрузите компьютер
- Проверьте: `python --version` в cmd

### "Не удалось создать виртуальное окружение"
```cmd
python -m pip install --upgrade pip
python -m pip install virtualenv
```

### "BOT_TOKEN not set"
- Откройте `.env` файл
- Добавьте строку: `BOT_TOKEN=ваш_токен_здесь`
- Сохраните файл

### "FFmpeg not installed"
- Запустите `install_ffmpeg.bat`
- Выберите способ установки
- Перезапустите бота

### Бот не запускается автоматически
- Откройте планировщик заданий (`taskschd.msc`)
- Найдите задачу `TikTokVideoBot`
- Проверьте путь к `start_bot.bat`
- Убедитесь, что задача включена

### Ошибки при скачивании
- Проверьте интернет-соединение
- Попробуйте другую ссылку
- Убедитесь, что ссылка корректная
