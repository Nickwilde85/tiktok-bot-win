# Changelog

## [Windows Edition] - 2026-04-28

### Добавлено
- ✅ Полная поддержка Windows 10/11
- 📦 `install.bat` - автоматическая установка
- 🎬 `install_ffmpeg.bat` - установка FFmpeg (4 способа)
- ▶️ `start_bot.bat` - быстрый запуск бота
- 🛑 `stop_bot.bat` - остановка бота
- 🔄 `setup_autostart.bat` - настройка автозапуска через Task Scheduler
- 📖 `README_WINDOWS.md` - подробная инструкция для Windows
- 🪟 Обновлен основной README с ссылкой на Windows-инструкцию
- 🏗️ `build_exe.bat` - сборка EXE файла
- 📚 `BUILD_EXE.md` - инструкция по созданию EXE
- 🔧 `setup_new_repo.bat` - настройка нового репозитория

### Изменено
- 📝 Обновлены команды установки для Windows в README
- 🔧 Адаптированы пути и команды для Windows окружения
- 🔗 Обновлена ссылка на репозиторий: tiktok-bot-win

### Удалено
- 🗑️ Удалены дублирующиеся файлы `tiktok-bot.service` из вложенных папок
- 🧹 Очищена структура проекта от ненужных вложенностей

### Технические детали
- Все bat-файлы используют кодировку UTF-8 (chcp 65001)
- Автозапуск настраивается через XML-конфигурацию Task Scheduler
- Проверка зависимостей перед запуском
- Автоматическая активация виртуального окружения
